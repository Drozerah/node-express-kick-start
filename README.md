Getting Started
---------------

```bash
# GET the latest version
λ git clone https://github.com/Drozerah/node-express-kick-start.git

# Change diretory
λ cd myproject

# Install NPM dependencies
λ npm install

# Start for development
λ npm run start:dev

# Start for production
λ npm start

# Start Eslint
λ npm run 
```
